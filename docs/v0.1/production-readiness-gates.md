# Production Readiness Gates (v0.1)

Production readiness is gated by `cubeshackles-integration` tests and
`scripts/check_v0_1_readiness.py`.

## Gate categories

1. Contract discipline
2. Authority boundaries (settlement, offline, validation, orchestration)
3. Secrets isolation (vault)
4. Disaster recovery planning
5. Chaos containment
6. Security controls
7. Operations doctrine
8. Angola pilot scope limits

Passing gates does **not** claim national deployment or regulator approval.
