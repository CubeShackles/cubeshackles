# Feature Freeze Doctrine (v0.1)

CubeShackles enters feature freeze when all v0.1 authority boundaries pass and
integration gates are green. After freeze:

- No new products
- No new tokenomics
- No new wallet features
- No new AI features
- Only bugs, security, performance, docs, and compliance fixes

## Freeze entry requires

- All v0.1 contracts pass
- All integration tests pass
- Settlement boundary pass
- Offline boundary pass
- Provincial topology pass
- Vault/secrets boundary pass
- Disaster recovery pass
- Chaos tests pass
- Security hardening pass
- Operations doctrine pass
- Pilot readiness docs complete

No fake production-ready claims.
